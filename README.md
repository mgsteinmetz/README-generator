## Description
This is a README generator that makes it easier for the user to create a README.md file for their projects!!

## Installation
To install, travel to my github, and download my code! 

## Usage 
To USE my code, all you have to do is type 'node index.js' into the integrated terminal!!

<img width="1405" alt="Screen Shot 2021-06-15 at 10 28 35 PM" src="https://user-images.githubusercontent.com/77464741/122153673-11a5ed80-ce29-11eb-8c35-630b015c1af9.png">

The user will then be prompted with various questions to successfully create a README.md file.

<img width="1403" alt="Screen Shot 2021-06-15 at 10 31 06 PM" src="https://user-images.githubusercontent.com/77464741/122153841-66496880-ce29-11eb-9c45-5e9e240f2380.png">

Once the user has successfully answered all the prompts, it will create a README.md file, and load it into your current directory!
